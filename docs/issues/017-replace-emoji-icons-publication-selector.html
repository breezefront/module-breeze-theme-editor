Issue 017 — Replace Emoji Icons in Publication Selector

Closed Task Area: publication-selector.html, _publication-selector.less

Background

The publication selector dropdown currently uses emoji characters as status/type indicators: 🟡 Draft, 🟢 Published, 📦 Publication item, ✓ active check, ✅ all-loaded, ⬇️ load more.

Emoji rendering is inconsistent across OS and browsers, cannot be styled with CSS (color, size, opacity), and looks out of place next to the Phosphor icon font already used throughout the toolbar. Replacing them with ph-* icons gives full CSS control and visual consistency.
